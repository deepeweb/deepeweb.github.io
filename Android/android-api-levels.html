Android API Levels in Hindi - (Version और Name) की पूरी जानकारी

Android API Level क्या होता है?

Android API Level एक यूनिक पहचान संख्या (Unique Identifier) होती है, जो किसी भी Android version के लिए दी जाती है। API Level से यह पता चलता है कि कोई Android डिवाइस या app किस version को support करती है। यह developers के लिए बहुत जरूरी होता है क्योंकि इसके माध्यम से वे यह तय कर सकते हैं कि उनकी app कौन-कौन से Android वर्शन पर चलेगी।
