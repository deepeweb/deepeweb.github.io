What is Android Runtime (ART) - Android Runtime (ART) क्या है?

Android Runtime:

Ad
  • Android Runtime (ART), Android का वो हिस्सा है जो आपकी app को चलाता है।
  • यह एक runtime environment है, जो Java/Kotlin में लिखे गए code को device पर चलने लायक बनाता है।
  • ART, Android 5.0 (Lollipop) से पहले Dalvik को replace करके लाया गया था।
  • ART में "Ahead-of-Time (AOT) compilation" का use होता है, जिससे app install होते समय ही उसका code native machine code में बदल दिया जाता है।
  • इससे app जल्दी और smooth चलती है, क्योंकि बार-बार compile नहीं करना पड़ता।
  • Dalvik में "Just-in-Time (JIT) compilation" होता था, जिसमें app हर बार चलने पर compile होती थी।
  • ART app की performance, memory management और battery life को बेहतर बनाता है।
  • ART garbage collection (unwanted memory clear करने) को भी बेहतर तरीके से करता है।
  • ART का काम है developer द्वारा लिखे गए bytecode को Android device पर execute करना।
  • यह पूरे Android system का एक जरूरी हिस्सा है, जिसके बिना कोई भी app चल नहीं सकती।
