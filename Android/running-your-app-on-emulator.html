Android Running app on Emulator in Hindi


Android Emulator पर App run करने का तरीका

अगर आप Android Studio में अपना app Emulator पर run करना चाहते हैं, तो नीचे दिए गए steps को follow करें:

Step 1: Android Emulator (AVD) setup करें

अगर आपने पहले से Emulator setup नहीं किया है, तो:

  • Android Studio खोलें।
  • "Tools" → "AVD Manager" पर जाएं।
  • "Create Virtual Device" पर click करें।
  • Device चुनें (जैसे Pixel 6, Nexus 5 आदि)।
  • सही Android API Level (System Image) download करें और Next दबाएं।
  • Emulator का नाम सेट करें और Finish दबाएं।

Step 2: App को Emulator पर run करें

  • Android Studio में अपना project खोलें।
  • Top Toolbar में "Device Selector" से अपना Emulator चुनें।
  • "Run" बटन दबाएं या Shift + F10 (Windows/Linux) | ⌘ + R (Mac) दबाएं।
  • Emulator start होगा और app load होने लगेगा।
  • अगर app सही तरीके से run हो गया, तो UI screen पर दिखेगा।

Step 3: अगर Emulator Slow है या Error आ रहा है

Performance Issues को Fix करें

  • RAM बढ़ाएं: AVD Settings में जाकर RAM को 2048MB या 4096MB करें।
  • Graphics Mode: Hardware Rendering चुनें (अगर supported हो)।
  • Cold Boot की जगह Quick Boot सेट करें।
  • Unnecessary Background Apps बंद करें ताकि ज्यादा RAM मिले।

Common Errors और उनके Solutions

HAXM Not Installed:

  • Intel HAXM को SDK Manager → SDK Tools में जाकर Install करें।

Device Not Booting:

  • AVD Manager में जाकर "Wipe Data" करके दोबारा run करें।

App Not Installed:

  • "Build → Clean Project" करें और फिर से run करें।
