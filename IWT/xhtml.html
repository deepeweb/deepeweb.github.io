Introduction to XHTML (Extensible HyperText Markup Language) in Hindi

XHTML

XHTML का पूरा नाम है Extensible HyperText Markup Language. यह HTML का एक updated और strict version है जिसे XML (Extensible Markup Language) के rules के अनुसार बनाया गया है।

Ad

HTML में कुछ flexibility होती है, जैसे tags को close न करना या uppercase letters का use करना। लेकिन XHTML में हर tag को सही तरीके से close करना जरूरी होता है, और सभी tags lowercase में ही होने चाहिए।

<!-- XHTML Example -->
<p>This is a paragraph.</p>
<br />

ऊपर के code में <br /> tag को self-close किया गया है, जो XHTML का requirement है। HTML में इसे केवल <br> भी लिखा जा सकता है, लेकिन XHTML में यह error माने जाएगा।

XHTML को XML की तरह treat किया जाता है। इसका मतलब है कि यह ज्यादा strict होता है और browser को इसे पढ़ने में किसी भी गलती पर दिक्कत हो सकती है।

XHTML का मुख्य उद्देश्य यह था कि web content को ज्यादा साफ़, structured और machine-readable बनाया जाए।

W3C (World Wide Web Consortium) ने XHTML को HTML की जगह पर promote किया था ताकि सभी websites में एक uniform structure हो सके।

हालांकि, आज के समय में HTML5 को ज्यादा use किया जाता है क्योंकि यह XHTML के मुकाबले ज्यादा flexible है। लेकिन XHTML अभी भी कुछ बड़े projects और पुराने systems में use होता है।

XHTML में attributes के value को हमेशा double quotes में लिखा जाता है, जैसे: type="text".

अगर आपकी website का code XHTML के rules को follow करता है, तो वो ज्यादा clean और future-compatible मानी जाती है।
