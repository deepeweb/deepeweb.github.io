HTML Text Formatting in Hindi

HTML में Text Formatting का मतलब होता है text को अलग-अलग style में दिखाना जैसे bold, italic, underline आदि। इसके लिए HTML में कई formatting tags दिए गए हैं जो browser को बताते हैं कि text कैसा दिखाना है।

HTML Headings

HTML में headings create करने के लिए <h1> से लेकर <h6> तक के tags का इस्तेमाल किया जाता है। ये tags text को heading के रूप में show करते हैं और हर tag की size अलग होती है।

<h1> सबसे बड़ी heading होती है और इसे page की main heading के लिए use किया जाता है। इसके बाद क्रमशः <h2>, <h3>... और <h6> तक heading की size छोटी होती जाती है।

Example

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

Headings ना सिर्फ design के लिए use होती हैं, बल्कि SEO (Search Engine Optimization) के लिए भी बहुत important होती हैं। Search engines headings को content के structure के रूप में पढ़ते हैं।

सही तरह से headings का इस्तेमाल करने से content को पढ़ना आसान हो जाता है और user को यह समझने में मदद मिलती है कि कौन सी जानकारी ज्यादा महत्वपूर्ण है।

<h1> tag एक page पर सिर्फ एक बार use करना चाहिए, जबकि बाकी headings को बार-बार use किया जा सकता है, जैसे <h2> या <h3>.

Headings को CSS के ज़रिए customize भी किया जा सकता है। आप color, font-size, alignment आदि आसानी से change कर सकते हैं।

HTML Formatting Elements

<b> – Bold Text

<b> tag किसी text को bold करने के लिए use होता है। यह सिर्फ appearance बदलता है, meaning नहीं। Example:

<p>This is <b>bold</b> text.</p>

<strong> – Important Text

<strong> tag भी text को bold करता है लेकिन यह semantic meaning देता है कि यह text ज़्यादा important है।

<p>This is <strong>very important</strong> text.</p>

<i> – Italic Text

<i> tag किसी text को italic style में दिखाता है। इसका use emphasis या stylish text के लिए किया जाता है।

<p>This is <i>italic</i> text.</p>

<em> – Emphasized Text

<em> tag भी italic दिखता है, लेकिन इसका मतलब होता है कि उस text पर जोर देना है। यह meaning को highlight करता है।

<p>This is <em>emphasized</em> text.</p>

<mark> – Marked/Highlighted Text

<mark> tag किसी text को highlight करने के लिए होता है, जैसे आप marker से कोई line mark करते हैं।

<p>This is <mark>highlighted</mark> text.</p>

<small> – Small Text

<small> tag text को छोटा कर देता है। यह पुराने या less important text दिखाने के लिए use होता है।

<p>This is <small>small</small> text.</p>

<del> – Deleted Text

<del> tag से ऐसा text दिखाया जाता है जो पहले था लेकिन अब हटा दिया गया है। यह usually line-through style में दिखता है।

<p>This is <del>deleted</del> text.</p>

<ins> – Inserted Text

<ins> tag ऐसा text दिखाता है जिसे नया insert किया गया हो। यह usually underline में दिखता है।

<p>This is <ins>inserted</ins> text.</p>

<sub> – Subscript Text

<sub> tag से text नीचे की ओर छोटा दिखता है। इसे mostly chemical formulas में use किया जाता है।

<p>H<sub>2</sub>O</p>

<sup> – Superscript Text

<sup> tag से text ऊपर की ओर छोटा दिखता है। इसे math और powers में use किया जाता है।

<p>2<sup>4</sup> = 16</p>


HTML Example

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>HTML Formatting Elements</title>
</head>
<body>
  <h1>HTML Formatting Elements Demo</h1>
  <p>This is <b>bold</b> text.</p>
  <p>This is <strong>very important</strong> text.</p>
  <p>This is <i>italic</i> text.</p>
  <p>This is <em>emphasized</em> text.</p>
  <p>This is <mark>highlighted</mark> text.</p>
  <p>This is <small>small</small> text.</p>
  <p>This is <del>deleted</del> text.</p>
  <p>This is <ins>inserted</ins> text.</p>
  <p>H<sub>2</sub>O</p>
  <p>2<sup>4</sup> = 16</p>
</body>
</html>
