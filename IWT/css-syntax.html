CSS Syntax

CSS syntax बताता है कि style को HTML element पर कैसे apply किया जाता है। इसमें तीन main parts होते हैं: Selector, Property, और Value.

Selector वह हिस्सा होता है जो बताता है कि किस HTML element पर style apply करनी है। जैसे अगर आप किसी

tag को color देना चाहते हैं, तो selector होगा p.

Selector के बाद curly braces {} के अंदर एक या ज़्यादा properties दी जाती हैं। हर property किसी एक design feature को control करती है, जैसे color, font-size, margin, आदि।

हर property के साथ एक value दी जाती है जो बताती है कि उस property को कैसे apply किया जाए। Property और value के बीच colon : और हर style rule के बाद semicolon ; लगाया जाता है।

Example of CSS syntax:

p {
  color: blue;
  font-size: 18px;
}

ऊपर दिए गए code में p selector है, color और font-size properties हैं, और उनके लिए blue और 18px values हैं।

CSS syntax बहुत simple है लेकिन spacing, brackets और semicolons का सही use करना जरूरी होता है ताकि browser उसे सही से समझ सके।

Multiple properties एक ही selector के लिए curly braces के अंदर लिखी जा सकती हैं, जिससे HTML elements को एक से ज़्यादा style दिए जा सकते हैं।
