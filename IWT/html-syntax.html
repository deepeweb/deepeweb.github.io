HTML Syntax in Hindi

HTML Syntax क्या होता है?

HTML Syntax का मतलब है कि HTML code को लिखने का सही तरीका। इसमें हर चीज को एक खास format में लिखा जाता है, जिससे browser समझ सके कि किसका क्या काम है।

HTML में content को tags से घेरा जाता है। हर tag angle brackets < > में लिखा जाता है। ज्यादातर tags के दो पार्ट होते हैं – Opening Tag और Closing Tag.

  • Opening Tag बताता है कि कौन सा element शुरू हो रहा है।
  • Closing Tag वही element खत्म करता है और इसमें / का use होता है।

Example के लिए: <p>This is a paragraph.</p>

यहां <p> opening tag है और </p> closing tag है।

HTML Syntax Example

Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page
  </head>
  <body>
    <h1>Welcome to My Website
    <p>This is a simple HTML example.

    <a href="https://www.google.com">Visit Google
  </body>
</html>

इस Example का मतलब:

  • <!DOCTYPE html> – यह बताता है कि यह HTML5 document है।
  • <html> – पूरा HTML document इसी tag में लिखा जाता है।
  • <head> – इसमें meta information और title रखा जाता है।
  • <title> – Browser tab का नाम दिखाता है।
  • <body> – यहां असली content आता है जो user को दिखता है।
  • <h1> – Main heading होती है।
  • <p> – Paragraph लिखा जाता है।
  • <a> – Hyperlink बनाता है।
