HTML Comments and Horizontal Lines in Hindi

HTML Comments

HTML Comments ऐसे notes होते हैं जो code में लिखे जाते हैं लेकिन web page पर दिखाई नहीं देते। Comments का इस्तेमाल code को समझाने या future reference के लिए किया जाता है।

HTML में comment लिखने का syntax होता है:

<!-- This is a comment -->

Comments browser में render नहीं होते यानी user को वो दिखाई नहीं देते। लेकिन developer जब code को पढ़ता है, तो comments उसकी मदद करते हैं।

Comments का use debugging में भी किया जाता है। आप किसी code को temporarily हटाने के लिए उसे comment कर सकते हैं बिना delete किए। Example:

<!-- <p>This paragraph is commented out and won't be shown.</p> -->

Horizontal Line in HTML

Horizontal line एक सीधी लाइन होती है जो content को अलग-अलग sections में divide करने के लिए इस्तेमाल होती है।

HTML में horizontal line बनाने के लिए <hr> tag का इस्तेमाल किया जाता है। यह self-closing tag होता है, यानी इसे बंद करने की जरूरत नहीं होती। Example

<p>This is paragraph one.</p>
<hr>
<p>This is paragraph two.</p>

ऊपर के example में <hr> tag दो paragraphs के बीच एक सीधी लाइन बना देगा जो visually दोनों को अलग कर देगा।

CSS की मदद से <hr> tag को design भी किया जा सकता है जैसे color, width, thickness आदि।
