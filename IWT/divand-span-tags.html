HTML Divand (<div>) and Span (<span>) Tags in Hindi

<div> Tag

<div> का मतलब होता है "division" या section। ये tag HTML page में एक block create करता है जिसमें आप multiple elements रख सकते हैं। यह एक block-level element होता है, यानी यह अपने लिए पूरी width लेता है और एक नई लाइन से शुरू होता है।

Example

<div>
  <h2>About Us</h2>
  <p>This is our company introduction.</p>
</div>

Web page को अलग-अलग हिस्सों में बांटने के लिए <div> बहुत useful होता है, खासकर जब आप CSS से अलग-अलग हिस्सों को style देना चाहते हैं।

<span> Tag

<span> एक inline element होता है, जिसका मतलब है कि यह text के बीच में use किया जाता है और यह नई लाइन पर नहीं आता। जब आपको text के किसी छोटे हिस्से को style करना हो, तब <span> tag का use किया जाता है।

Example

<p>This is a <span style="color: red;">red</span> word.</p>

यहां सिर्फ "red" word का color change होगा, बाक़ी paragraph वैसा का वैसा रहेगा।

फर्क <div> और <span> में

  • <div> एक block-level element है जो पूरे section को cover करता है।
  • <span> एक inline element है जो सिर्फ text के हिस्से पर apply होता है।
  • <div> का इस्तेमाल layout design और structure के लिए किया जाता है।
  • <span> का इस्तेमाल text को style या highlight करने के लिए किया जाता है।

जब CSS use करनी हो तो <div> और <span> को class या id के साथ use करना बहुत फायदेमंद होता है। इससे आप easily custom styles apply कर सकते हैं।
