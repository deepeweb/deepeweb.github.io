CSS Implementation (Inline, Internal, External) in Hindi

CSS को HTML में तीन तरीकों से implement किया जा सकता है: Inline CSS, Internal CSS, और External CSS। ये तीनों methods अलग-अलग situations में use किए जाते हैं।

1. Inline CSS

Inline CSS तब use किया जाता है जब आप किसी एक HTML element को directly style देना चाहते हैं। इसमें style attribute का use किया जाता है और CSS code उसी element के अंदर लिखा जाता है।

Example:

<p style="color: red; font-size: 18px;">This is an inline styled paragraph.</p>

यह method quick styling के लिए useful होती है, लेकिन बड़ी websites के लिए maintain करना मुश्किल होता है। यह सिर्फ उसी element पर apply होती है, बाकी किसी पर नहीं।

2. Internal CSS

Internal CSS एक ही HTML file के अंदर use होती है। इसे <head> tag के अंदर <style> tag में लिखा जाता है। अगर किसी single page को style करना हो तो internal CSS अच्छा option है।

Example:

<head>
  <style>
    h1 {
      color: blue;
      text-align: center;
    }
  </style>
</head>

इस method में सभी styles एक ही HTML file में लिखे जाते हैं, जिससे changes करना आसान होता है लेकिन performance पर थोड़ा असर पड़ सकता है।

3. External CSS

External CSS सबसे ज्यादा use किया जाने वाला और recommended तरीका है। इसमें CSS code एक अलग .css file में लिखा जाता है और HTML में <link> tag के जरिए जोड़ा जाता है।

Example (HTML में):

<head>
  <link rel="stylesheet" href="style.css">
</head>

style.css file:

body {
  background-color: #f0f0f0;
  font-family: Arial;
}

External CSS का सबसे बड़ा फायदा यह है कि आप एक ही file से पूरी website को control कर सकते हैं। यह साफ-सुथरा, fast और easy to manage होता है।
