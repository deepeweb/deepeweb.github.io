Adding jQuery to web page:

jQuery को अपनी website में जोड़ना बहुत आसान होता है। इसके लिए आपको सिर्फ jQuery की library को HTML page में include करना होता है। आप दो तरीके से jQuery जोड़ सकते हैं – CDN (Content Delivery Network) से या local file के जरिए।

CDN से jQuery जोड़ने के लिए आपको jQuery की hosted file का link अपनी HTML file के <head> या <body> section में <script> tag के अंदर लगाना होता है।

Example के लिए, आप Google CDN से jQuery ऐसे जोड़ सकते हैं:

इस method का फायदा यह है कि file already cached हो सकती है user के browser में, जिससे आपकी website जल्दी load होती है।

अगर आप jQuery को locally use करना चाहते हैं तो सबसे पहले jQuery की .js file को jQuery की official site से download करें। फिर उस file को अपने project folder में रखें और <script src="your-folder/jquery.min.js"></script> tag से जोड़ दें।

jQuery को <script> tag के ज़रिए हमेशा किसी भी custom JavaScript या jQuery code से पहले load करना जरूरी होता है, वरना $ और jQuery functions काम नहीं करेंगे।

अगर आपकी HTML file के अंत में jQuery add करते हैं (जैसे कि </body> tag से पहले), तो यह performance के लिए अच्छा होता है क्योंकि तब page elements पहले load हो जाते हैं।
