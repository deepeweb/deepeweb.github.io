JavaScript Function and Function Declaring in Hindi

JavaScript Functions

JavaScript में function एक reusable block of code होता है, जिसे बार-बार use किया जा सकता है। Function तब run होता है जब उसे call किया जाता है। इससे code छोटा, organized और manageable बनता है।

Function का use हम तब करते हैं जब किसी task को बार-बार करना हो, जैसे calculation, data process करना, या किसी value को return करना। इससे हर बार वही code दोहराने की जरूरत नहीं पड़ती।

JavaScript में function को declare करने के लिए function keyword का use किया जाता है। उसके बाद function का नाम, parentheses () और फिर curly brackets {} के अंदर code लिखा जाता है।
