jQuery Selectors in Hindi

jQuery Selectors:

jQuery Selectors का use किसी HTML element को select करने के लिए किया जाता है ताकि उस पर कोई action apply किया जा सके। ये selectors बहुत powerful होते हैं और JavaScript के मुकाबले बहुत simple syntax में काम करते हैं।

सबसे basic selector होता है ID selector. इसका use किसी particular element को select करने के लिए किया जाता है जिसकी unique ID हो। इसे लिखते हैं $("#idName") की तरह। जैसे $("#title") का मतलब है id="title" वाले element को select करना।

Types of jQuery Selectors in Hindi

1. Basic Selectors

(a) ID Selector –

किसी element को उसकी id से select करने के लिए

// JavaScript
$("#myId")
<!-- HTML -->
<p id="myId">Hello</p>

(b) Class Selector –

किसी element को उसकी class से select करने के लिए

// JavaScript 
$(".myClass")
<!-- HTML -->
<div class="myClass">Welcome</div>

(c) Element Selector –

सीधे किसी tag को select करने के लिए

// JavaScript
$("p")
<!-- HTML -->
<p>This is a paragraph.</p>

2. Universal Selector

सभी HTML elements को select करता है।

// JavaScript
$("*")
<!-- HTML -->
<p>Hello</p>
<div>World</div>

इस example में <p>, <div> दोनों select हो जाएंगे।

3. Group Selector

Multiple elements को एक साथ select करता है।

// JavaScript
$("h1, p, .box")
<!-- HTML -->
<h1>Heading</h1>
<p>Paragraph</p>
<div class="box">Box Content</div>

4. Attribute Selector

Attribute की help से element select करता है।

(a) Specific attribute

$("input[type='text']")

(b) Attribute starts with (^=)

$("a[href^='https']")

(c) Attribute ends with ($=)

$("img[src$='.png']")

(d) Attribute contains (*=)

$("div[class*='box']")

5. Hierarchy Selectors

(a) Descendant Selector (space)

$("div p")

किसी <div> के अंदर मौजूद सभी <p> elements select होंगे।

(b) Child Selector (>)

$("ul > li")

Direct child <li> elements select होंगे।

(c) Adjacent Sibling (+)

$("h2 + p")

ऐसे <p> जो किसी <h2> के तुरंत बाद आएं।

6. Form Selectors

(a) All input fields

$(":input")

(b) Checked elements

$(":checked")

(c) Selected option in dropdown

$("select option:selected")

7. jQuery Filters (Extra)

(a) First element

$("p:first")

(b) Last element

$("div:last")

(c) Even elements

$("li:even")

(d) Odd elements

$("li:odd")
