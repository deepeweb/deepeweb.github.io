Bootstrap Forms in Hindi

Bootstrap Forms

Forms का use user से data collect करने के लिए होता है (जैसे login, signup, feedback आदि)। Bootstrap में forms को clean, responsive और stylish banाना बहुत आसान है।

1. Basic Form

HTML

form-control class input fields को full width aur responsive बनाती है।

form-label class label ko properly style करती है।

2. Form with Checkbox & Radio

HTML

form-check-input और form-check-label checkbox/radio को style करते हैं।

3. Inline Form (One Line Form)

HTML

d-flex class form को horizontally align करती है।

4. Form with Select Dropdown

HTML

form-select class dropdown को styled बनाती है

5. Floating Label Form

HTML

Input field पर text अंदर float होकर ऊपर चला जाता है।

6. Disabled & Readonly Inputs

HTML

Disabled → user type नहीं कर सकता।

Readonly → user देख सकता है but edit नहीं कर सकता।

In Short:

  • .form-control → input, textarea, select fields के लिए
  • .form-label → labels के लिए
  • .form-check → checkbox/radio groups
  • .form-select → dropdowns
  • .form-floating → floating labels
  • .d-flex → inline form
