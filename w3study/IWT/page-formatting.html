HTML Page Formatting in Hindi

HTML में Page Formatting का मतलब होता है किसी web page के content को अच्छे से दिखाना – जैसे text को bold करना, italic में दिखाना, underline करना, बीच में लाना या list में दिखाना।

Formatting करने के लिए HTML में कुछ special tags होते हैं जिन्हें Formatting Tags कहते हैं। ये tags browser को बताते हैं कि किसी content को कैसे display करना है।

Adding a New Paragraph

HTML में नया paragraph बनाने के लिए <p> tag का इस्तेमाल किया जाता है। जब आप कोई नया paragraph बनाना चाहते हैं, तो content को <p> और </p> के बीच लिखते हैं। Example:

Adding a Line Break

अगर आप एक ही paragraph के अंदर कहीं पर नया line शुरू करना चाहते हैं, तो <br> tag का use करते हैं। यह self-closing tag होता है और सिर्फ एक line break देता है।

HTML

यह दोनों lines को अलग-अलग दिखाता है लेकिन उन्हें एक ही paragraph में रखता है।

Inserting a Blank Space

HTML में एक से ज्यादा space देने पर browser उन्हें ignore कर देता है। Extra blank space देने के लिए &nbsp; (Non-Breaking Space) का use किया जाता है।

HTML

हर &nbsp; एक space के बराबर होता है। आप जितने चाहो, उतने लगाकर spacing बढ़ा सकते हो।

Changing Page Background

HTML में page का background बदलने के लिए CSS का use होता है। आप या तो background color set कर सकते हो या image लगा सकते हो। यह style या <body> tag में bgcolor attribute के ज़रिए किया जा सकता है।
