HTML Events and Calling JavaScript Functions on Event

HTML में event वह होता है जब user किसी element के साथ interact करता है, जैसे button पर click करना, mouse ले जाना, keyboard key दबाना, form submit करना आदि। JavaScript की मदद से हम इन events पर response दे सकते हैं।
