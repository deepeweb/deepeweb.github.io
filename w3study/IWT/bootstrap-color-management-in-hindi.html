Bootstrap Color Management

Bootstrap में colors को manage करने के लिए पहले से ही predefined color utility classes दी गई हैं। इनका use करके आप text, background, border और buttons आदि को आसानी से style कर सकते हो।

1. Text Colors

Bootstrap में text color change करने के लिए .text-* classes use होती हैं।

Bootstrap 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Example</title>
  <!-- Bootstrap 5 CDN -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
<p class="text-primary">This is Primary Text</p>
<p class="text-success">This is Success Text</p>
<p class="text-danger">This is Danger Text</p>
<p class="text-warning">This is Warning Text</p>
<p class="text-info">This is Info Text</p>
<p class="text-dark">This is Dark Text</p>
<p class="text-muted">This is Muted Text</p>

  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js">
</body>
</html>

5. Color Shades

Bootstrap 5 में colors के shades (100 से 900 तक) भी available हैं।

Bootstrap 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Example</title>
  <!-- Bootstrap 5 CDN -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
<div class="p-3 mb-2 bg-primary text-white">bg-primary</div>
<div class="p-3 mb-2 bg-primary-subtle text-dark">bg-primary-subtle</div>
<div class="p-3 mb-2 bg-success-subtle text-dark">bg-success-subtle</div>
<div class="p-3 mb-2 bg-danger-subtle text-dark">bg-danger-subtle</div>

  <!-- Bootstrap JS Bundle -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js">
</body>
</html>

In short:

  • Text colors → .text-*
  • Background colors → .bg-*
  • Button colors → .btn-*
  • Border colors → .border-*
  • Shades → *-subtle
