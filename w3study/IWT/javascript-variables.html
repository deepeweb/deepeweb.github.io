JavaScript Variables in Hindi

JavaScript variable:

JavaScript में variable एक container होता है जिसमें हम कोई value store करके बाद में use कर सकते हैं। जैसे आप एक box में कुछ रखकर ज़रूरत पड़ने पर निकालते हैं, उसी तरह variable भी काम करता है।

JavaScript में variables को declare करने के लिए तीन keywords use होते हैं: var, let, और const। ये तीनों variables को define करने के तरीके हैं, लेकिन इनके rules थोड़े अलग होते हैं।

1. var keyword

var पुराने JavaScript versions में use होता था। इससे declared variable को हम दुबारा redeclare भी कर सकते हैं और उसकी value को change भी कर सकते हैं।

JavaScript

2. let keyword

let ES6 (modern JS) में introduce हुआ। इससे declared variable को दोबारा redeclare नहीं कर सकते, लेकिन उसकी value को change किया जा सकता है।

JavaScript

3. const keyword

const से declare किया गया variable constant होता है यानी उसकी value को change नहीं किया जा सकता। यह mostly उन values के लिए use होता है जो कभी बदलनी नहीं चाहिए।

JavaScript

Variables में हम अलग-अलग types की values store कर सकते हैं। JavaScript में कुछ main types हैं: String, Number, Boolean, Object, Array, और Undefined.
