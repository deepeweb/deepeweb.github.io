Bootstrap Images

Bootstrap में images को handle करने के लिए बहुत सारी predefined classes दी गई हैं, जिनसे images responsive, rounded, circular, thumbnail वगैरह बन जाती हैं। Bootstrap images को resize, shape और align करने के लिए utility classes provide करता है।

1. Responsive Image

.img-fluid class use करने से image हमेशा parent container की width के हिसाब से adjust हो जाती है।

HTML

यह image automatically small screen पर shrink हो जाएगी।

2. Rounded Corners

Image को rounded बनाने के लिए .rounded class use होती है।

HTML

3. Circle Image

Completely circular image बनाने के लिए .rounded-circle class use होती है।

HTML

Mostly profile photos के लिए use होती है।

4. Thumbnail Image

Image को bordered thumbnail format में दिखाने के लिए .img-thumbnail class use होती है।

HTML

5. Image Alignment

Bootstrap की utility classes use करके images को align किया जा सकता है।

HTML
  • float-start → Left align
  • loat-end → Right align
  • mx-auto d-block → Center align

6. Image in Figure (with Caption)

HTML

figure tag use करने से image के साथ caption add किया जा सकता है।
