JavaScript Control Flow in Hindi (Statements and Loops)

JavaScript में Control Flow का मतलब होता है कि हमारा code किस order में चलेगा। जब हम कोई program लिखते हैं, तो उसमें कई statements होती हैं जिन्हें एक order में execute किया जाता है।

By default, JavaScript code top to bottom line-by-line चलता है। लेकिन हम control flow statements का use करके यह decide कर सकते हैं कि कौन-सी line कब और कितनी बार चलेगी।

Control Flow को manage करने के लिए JavaScript में तीन main categories होती हैं: Conditional Statements, Loops, और Jump Statements.

Conditional Statements

JavaScript में Conditional Statements का use decision लेने के लिए किया जाता है। इसका मतलब है – किसी condition (शर्त) के true या false होने के अनुसार अलग-अलग code run करना।

Conditional statements की मदद से आप अपनी website या program को smart बना सकते हैं। जैसे – अगर user की age 18 से ज्यादा है तो "Adult" दिखाओ, वरना "Minor".

JavaScript में commonly तीन तरह की conditional statements होती हैं: if, if...else, और if...else if...else। इनके अलावा एक shortcut होता है जिसे ternary operator कहते हैं।

1. if Statement

if statement का use तब किया जाता है जब हमें कोई काम सिर्फ एक condition true होने पर करना हो।

JavaScript

2. if...else Statement

अगर हमें दो conditions के बीच decision लेना हो – एक true पर और दूसरी false पर – तो if...else use किया जाता है।

JavaScript

3. if...else if...else Statement

जब multiple conditions check करनी हो, तब हम if...else if...else structure use करते हैं।

JavaScript

4. Ternary Operator

यह एक short form है if...else का। इसे single line में condition check करने के लिए use किया जाता है।

JavaScript

Conditional statements की मदद से आप अपने program के flow को control कर सकते हैं। यह real-world decisions को code में convert करने का सबसे आसान तरीका है।


Loops

JavaScript में loops का इस्तेमाल तब किया जाता है जब हमें एक ही code को बार-बार run करना होता है। इससे code छोटा, साफ और efficient बनता है।

Loops के ज़रिए हम किसी काम को तब तक repeat कर सकते हैं जब तक कोई specific condition true रहती है। जैसे 1 से 10 तक numbers print करना, या किसी array के सारे items को check करना।

JavaScript में कई प्रकार के loops होते हैं: for, while, do...while, और for...of, for...in। हर loop का अपना use case होता है, लेकिन सभी का main काम repetition करना है।

1. for loop

यह सबसे common loop है, जब आपको पता हो कि कितनी बार loop चलाना है।

JavaScript

2. while loop

जब आपको पहले condition check करनी हो और फिर loop चलाना हो, तो while loop use किया जाता है।

JavaScript

3. do...while loop

यह loop कम से कम एक बार जरूर चलता है, चाहे condition false हो। पहले code run होता है, फिर condition check होती है।
